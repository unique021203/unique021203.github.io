Linux入门级指令

ls 查看该文件夹下的文件

ls -a 显示隐藏文件

ls -l 查看文件的详细信息

touch 新建文件

mkdir 新建文件夹

cd 跳转到…

cd .. 返回上级目录

cd /绝对路径/ 跳转到绝对路径

pwd 查看当前的绝对路径

rmdir 删除空目录

rm 删除非空目录

-i 删除前询问是否删除

-f 强制删除，不询问

-r 递归删除

cp [选项] 源文件.. 目的地 复制

-r 递归复制

mv [选项] 源文件.. 目的地 移动

mv oldname newname 文件重命名

echo 复读机

man 具体指令 可以看指令的介绍

Linux进阶指令

$ find -name filename

递归查找所有符合filename 的文件并输出路径

$ grep [选项] [要查找的字符串] [文件名or目录名]

在指定的文件or目录中寻找含有指定字符串的行，并输出到屏幕上，如果没有指定目标文件，则到标准输入上找

-i 忽略大小写

-r 文件夹递归查找

-n 显示行号

-v 反向查找（查找没有指定字符串的）

$ tree [选项] [目录名]

# 根据文件目录生成文件树

-a 列出全部文件

-d 只列出目录

$ locate [选项] [目录名]

# 与find类似，但find是去硬盘里找，locate是去数据库# 里找，locate的速度更快，但不是实时的，查找结果以# 数据库的更新为准

$ chmod [权限设定字串] [文件]

[权限设定字串]格式：

[ugoa…] [ [+-=] [rwxX] [,…]]

u: 拥有者；g: 与拥有者同组的人；o: 其他；a: 全部

+：增加权限；-：减少权限；=：唯一设定权限

r：可读取；w：可写入；x：可执行

$ chmod abc file

# abc三个数字分别表示ugo，r=4, w=2, x=1，数字加和表示权限

eg:

$ chomd 777 file

$ chmod a=rwx file

二者效果相同

作业中最常用的就是

diff [选项] file1 file2

检查两个文件是否相同

-b 不检查空格

-B 不检查空行

-q 仅显示有无差异，不显示详细信息

sed [选项] ‘命令’ 输入文本

选项：

-n 安静模式，仅输出经过sed处理的内容

-e 进行多项编辑

-i 直接修改档案内容

命令：

a: 在当前行的下一行添加一行

$ sed ‘2a drink tea’ my.txt

# 在第二行后添加drink tea

c: 用新文本取代本行文本

$ ‘2,5c No 2-5 Number’ my.txt

# 将2-5行的内容取代成2-5Number

i： 在当前行的上一行添加一行

d: 删除当前行内容

$ sed ‘2d’ my.txt

# 删除第二行

p: 显示，把选择的内容输出

$ sed -n ‘5,7p’ my.txt

# 显示5-7行的内容（-n 与 -p 一般搭配使用）

s: s/re/string，re为正则表达式，string为字符串，将正则表达式替换为字符串

$ sed ‘s/old/new/g’ my.txt

awk [选项参数] ‘script’ filename

$ awk ‘$1>2 {print $1,$3}’ my.txt

命令中出现的$n代表一行中用空分隔后的第n项

# 文件中所有第一项>2的行，输出第一项和第三项

-F 用于指定分割的字符,模式是空格

$ amk -F , ‘{print $2}’ my.txt

# 使用’,’分割，输出第二项

$ awk -F ‘[ ,]’ ‘{print $1,$2,$5}’ log.txt

# 使用多个分隔符.先使用空格分割，然后对分割结果再使用”,”分割（把多个分割符用[]括起来，按顺序排列）

Vim简单使用

  1. vim filename进入vim；
  2. i进入编辑模式；
  3. 不可以使用鼠标操作，平时使用的删除键会删除光标的前一个字符，del键会删除光标现在所在处的字符；
  4. Esc+:进入底线命令模式；
  5. q不保存退出，w保存，wq保存并退出；

GCC编译生成可执行文件

先来看涉及的到的文件后缀：

file.c 常规c源码

file.i 不需要预处理的c源码

file.h 头文件

file.s 汇编语言文件

file.S 必须进行预处理的汇编语言文件

file.o 目标文件（能直接被计算机识别的二进制代码）

gcc的选项和涉及到的文件类型是密切相关的，注意选项要单独写，-p -q，不能写成-pq。

语法：

gcc [选项…] [参数…]

选项：

-c: 只编译不链接

$ gcc -c test.c

#将test.c编译成test.o

-o: 生成指定的输出文件

$ gcc test.c -o test

#生成名为test的可执行文件

$ gcc test.c

#默认生成名为a.out的可执行文件

-I PATH : 编译时指定头文件的目录，使用相对路径

$ gcc -c -I ../include/ main.c

#main.c中使用了自定义的头文件，头文件放在上级目录的include文件夹中

#使用标准库时不需要指定目录，如<stdio.h>；使用自定义的头函数时需要指定目录，如”fibo.h“

一点小问题：

起初完成lab0的课下时，未正确使用-I选项，查到了VPATH这种办法，但还没有完全弄明白，VPATH似乎只能给出依赖文件里的路径，和vpath还有一些区别，但如果能通过VPATH统一设置Makefile查找头文件时的路径，在工程中就能大大简化gcc命令行。

-Wall 显示最多的警告信息

-S 将C代码转换为汇编代码

-N 列出依赖

一点小问题：

-N是干嘛的，为什么打上去没有反应

关于gcc多文件编译的问题：

  1. 若写在一条语句里需要注意先后顺序的问题
  2. 若分语句写注意执行Makefile里的gcc -c 是倒着执行的

Makefile入门

首先要理解Makefile其实就是把很多条指令写到了一个叫做Makefile的文件里，所以可以直接在黑框里打的指令都是可以直接敲在Makefile里的，只是在黑框里敲一次只能敲一条，在Makefile里一下可以打好多条，类似联机批处理和脱机批处理。

Makefile的格式如下：

Tag: 列出依赖文件

/tab/ gcc ….

​ …

# 指令的开头一定要敲一个tab，空格是不可以的

# Tag可以是要生成的目标文件，也可以只是一个

# name，它的作用就是在黑框里敲make Tag

Tag1: …

构建好Makefile可以直接输入make Tag，则将执行该Tag后的语句；如果只输入make，则找到Makefile中的第一个Tag并执行。

Shell 入门

  • 定义变量

    定义变量时，变量名不加$，变量名和等号之间不能有空格

  • 使用变量

    使用一个定义过的变量，只要在变量名前添加$符号即可

  • shell 字符串

    • 单引号：

      • 单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的；
      • 单引号中不能出现单独的单引号（使用转义字符也不可以），但可以出现成对的单引号作为字符串拼接使用；

    • 双引号

      • 双引号里可以有变量

      • 双引号里可以出现转义符

  • shell 数组

    • 定义数组

      用括号来表示数组，数组元素用空格分开

    • 读取数组

      使用@可以获取数组中的所有元素

  • shell 传递参数

    • 在执行shell脚本时可以向脚本传递参数，脚本内获取参数的格式为$n，n代表传入的第几个参数。

    • 几个特殊字符：

      $#：传递到脚本的参数个数

      $*以一个字符串显示传递的全部参数

  • shell基本运算符

​ 原生bash 不支持数学运算，但可以借助expr实现。

​ 需要注意以下几点：

  1. 表达式和运算符之间要有空格1 + 1 ；

  2. 完整的表达式要被``包含；

  3. 条件表达式要放在[]之间，并且要有空格，[ $a == $b ]，[]和变量之间要有空格，==和变量之间也要有空格。

    • 算术运算符

      + - * / % = == !=

      注意*前必须要加\才能实现乘法运算。

    • 关系运算符

      -eq: euqal

      -ne: not equal

      -gt: greater than

      -lt: less than

      ge: greater than or equal

      le: less than or equal

    • 布尔运算符

      !: 非

      -o: 或

      -a: 与

    • 逻辑运算符

      && : 逻辑AND

      ||: 逻辑OR

    • 字符串运算符

      =: 检测两个字符串是否相等

      !=: 检测两个字符串不相等

      -z: 检测字符串长度是否为0

      -n: 检测字符串长度是否不为0

      $: 检测字符串是否为空

  • shell printf

    • %s %c %d %f

    • %-10s%一个宽度为10个字符，-表示左对齐，没有则表示右对齐；

    • %-4.2f 指格式化为小数，其中 .2 指保留2位小数。

    • if else
